# 快速上手指南

## 🚀 5分钟快速开始

### 1. 安装扩展（1分钟）

在 VS Code 扩展市场搜索 "i18n Hover Translator" 并安装，或直接安装 `.vsix` 文件。

### 2. 准备翻译文件（2分钟）

在项目根目录创建 `translations.txt`：

```txt
# 基础翻译
BTN_OK=确定
BTN_CANCEL=取消
MSG_SUCCESS=操作成功
ERROR_NETWORK=网络连接失败
```

### 3. 开始使用（2分钟）

```javascript
// 悬停查看翻译
Language.GetContent("BTN_OK");  // 显示：确定

// 中文反向查找
GameUtil.ShowTips("确定");  // 显示可用的key列表，点击即替换
```

完成！✅

## 🎯 核心功能速览

### 功能1：查看翻译（最基础）

```javascript
Language.GetContent("BTN_OK");  // 悬停 → 显示"确定"
```

### 功能2：中文反向查找（最实用）

```javascript
GameUtil.ShowTips("确定");  // 悬停 → 显示key列表 → 点击替换
```

### 功能3：强制模式（最灵活）

按 `Ctrl+Shift+H` 切换，可查看任意字符串的翻译。

## ⚙️ 可选配置

### 指定翻译文件路径

```json
{
  "i18nHover.translationFilePath": "D:/project/i18n/zh_CN.txt"
}
```

### 添加自定义函数

```json
{
  "i18nHover.functionPatterns": [
    "MyI18n\\.Get\\s*\\([^\"']*$"
  ]
}
```

## 💡 使用技巧

### 技巧1：重构硬编码中文

1. 找到硬编码的中文：`alert("操作成功")`
2. 悬停查看可用的key
3. 点击合适的key
4. 自动替换为：`alert("MSG_SUCCESS")`

### 技巧2：检查翻译覆盖率

1. 按 `Ctrl+Shift+H` 开启强制模式
2. 逐个检查代码中的字符串
3. 找出缺少翻译的key
4. 补充到翻译文件

### 技巧3：快速切换翻译key

1. 悬停在当前key上
2. 查看翻译内容是否正确
3. 如果不对，查看其他可用的key
4. 点击切换到正确的key
