## 💡 使用技巧

### 零配置使用
大多数情况下，你无需任何配置！只需：
1. 在项目根目录创建 `translations.txt`
2. 添加翻译内容（`key=value` 格式）
3. 开始使用

### 快速切换模式
按 `Ctrl+Shift+H` 在正常模式和强制查找模式之间切换。

### 一键替换中文
悬停在中文字符串上，点击建议的 key，自动替换！
